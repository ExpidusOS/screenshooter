Help for the Xfce screenshot tool

Introduction

This application enables you to take screenshots of the entire screen or of the active window. You can set the delay that elapses before the screenshot is taken and the action that will be done with the screenshot: save it to a PNG file, copy it to the clipboard, or open it using another application.

A plugin for the Xfce panel is also available.

Using the application

Via the user interface

To launch the application, use the Take a screenshot item in the Accessories category of Xfce's main menu.

This window will be displayed:

Area to capture

The Area to screenshot section enables you to set what will be screenshoted:

  • "Entire screen" takes a screenshot of screen as you see it, in its totality.
  • "Active window" takes a screenshot of the active window, that is to say the one which was active before the dialog to take the screenshot appeared.

Delay before taking the screenshot

The Delay before taking the screenshot section enables you to set the delay that will elapse between pressing the Take the screenshot button and taking the screenshot. This delay will enable to open menus or to do whatever action you want to be displayed on the screenshot.

Action

The Action section enables you to choose what should be done with the screenshot. Three choices are available.

Save

The Save option will save the screenshot to a PNG file. If you check the Save by default to checkbox, the screenshot will be saved after you press the Take the screenshot button, without showing any save dialog. Else, a save dialog will be displayed, also showing a preview of the screenshot, you will then be able to set the save location and the name of the file.

Copy to the clipboard

The Copy to the clipboard option enables you to paste the screenshot in another application, such as a word processor.

Open with

The Open with option saves the screenshot to the system's temporary directory and opens it with the application set using the dropdown list below. The applications which support images are automatically detected and added to the dropdown list.

Via command line

Les options de ligne de commande permettent de prendre des captures d'écran rapidement. Elles permettent aussi de faire fonctionner la touche Prt Scrn en haut à droite de la plus part des claviers, pour cela, réglez les raccourcis de votre environnement de bureau pour qu'il lance xfce4-screenshooter avec les bonnes options lorsque la touche Prt Scrn est appuyée.

Les options en ligne de commande

L'option -w

L'option -w permet de prendre une capture d'écran de la fenêtre active.

L'option -f

L'option -f permet de prendre une capture d'écran de l'écran tout entier.

L'option -d

L'option -d suivie d'un entier positif permet de régler le délai en secondes avant la prise de la capture d'écran.

L'option -s

L'option -s suivie d'un chemin vers un dossier existant permet de choisir où la capture d'écran est sauvegardée. Si vous choisissez d'afficher la fenêtre de sauvegarde (voir plus bas), elle s'ouvrira dans ce dossier. Sinon, la capture d'écran est directement sauvegardée dans ce dossier.

L'option -h

L'option -h permet de ne pas afficher la fenêtre de sauvegarde, et de sauvegarder les captures dans l'emplacement par défaut.

Utilisation de l'extension pour le tableau de bord de Xfce

Ajout

Pour ajouter l'extension au tableau de bord, cliquez droit sur le tableau de bord et sélectionnez Ajouter un élément. Dans la fenêtre qui apparaît, choisissez d'ajouter l'extension de capture d'écran.

Configuration

Vous pouvez ensuite configurer la prise des capture d'écran en cliquant droit sur l'icône de l'extension, sur le tableau de bord, et en choisissant Configurer. Une fenêtre de configuration similaire à la fenêtre de l'application que vous pouvez voir plus haut apparaît alors. Merci de regarder plus haut pour les différentes options.

Utilisation

Il suffit ce cliquer sur l'icône de l'extension pour prendre une capture d'écran.
