Bantuan untuk Xfce Screenshooter

Pengenalan

Aplikasi ini memungkinkan anda untuk menangkap seluruh layar, jendela yang aktif atau daerah terpilih. Anda dapat mengatur jeda waktu sebelum cuplikan layar diambil dan aksi yang akan dilakukan dengan cuplikan layar: simpan ke berkas PNG, salin ke papan klip atau buka dengan aplikasi lainnya.

Plugin untuk panel Xfce juga tersedia.

Menggunakan aplikasi

Via antarmuka pengguna

Untuk menjalankan aplikasi, gunakan item "Ambil Cuplikan Layar" pada kategori "Aksesoris" dari menu utama Xfce.

Daerah untuk ditangkap

Bagian "Daerah untuk ditangkap" menmungkinkan anda untuk mengatur dari apa cuplikan layar akan diambil:

  • "Seluruh layar" mengambil cuplikan layar dari seluruh layar seperti yang anda lihat.
  • "Jendela aktif" mengambil cuplikan layar dari jendela yang aktif, yaitu jendela yang aktif sebelum dialog ini muncul, atau jika anda mengatur jeda, adalah jendela yang aktif setelah jeda.
  • "Pilih daerah" memungkinkan anda untuk memilih daerah yang akan ditangkap dengan cara klik pada titik di layar tanpa melepas tombol tetikus, tarik tetikus anda ke sudut daerah, lalu lepaskan tombol tetikus.

Jeda sebelum mengambil cuplikan layar

Bagian "Jeda sebelum mengambil cuplikan layar" memungkinkan anda untuk mengatur jeda yang akan berlangsung mulai saat menekan tombol Ambil cuplikan layar sampai mengambil cuplikan layar. Jeda ini memungkinkan anda untuk membuka menu atau aksi apapun yang ingin anda tampilkan di cuplikan layar.

Aksi

Bagian "Aksi" memungkinkan anda untuk menentukan apa yang harus dilakukan dengan cuplikan layar. Terdapat tiga pilihan yang tersedia.

Simpan

Opsi "Simpan" akan menyimpan cuplikan layar ke berkas PNG. Jika anda mencentang kotak "Simpan sebagai standar ke", cuplikan layar akan disimpan ke folder di sebelah kanan setelah anda menekan tombol "Ambil cuplikan layar", tanpa menampilkan dialog simpan apapun. Sebaliknya, dialog simpan akan ditampilan, yang juga menampilkan pratayang cuplikan layar. Anda selanjutnya akan dapat mengatur lokasi penyimpanan, dan nama berkas.

Salin ke papan klip

Opsi "Salin ke papan klip" memungkinkan anda untuk merekatkan cuplikan layar di aplikasi lain, seperti pemroses kata. Menghentikan aplikasi sebelum cuplikan layar direkatkan sering menghapus citra dari papan klip. Beberapa manajer papan klip memungkinkan anda untuk mencegah hal ini.

Buka dengan

Opsi "Buka dengan" menyimpan cuplikan layar ke direktori sementara sistem dan membukanya dengan seperangkat aplikasi menggunakan senarai di bawah. Aplikasi yang menyokong citra akan otomatis terdeteksi dan ditambahkan ke senarai.

Via baris perintah

Opsi baris perintah memungkinkan anda mengambil cuplikan layar secara cepat. Opsi ini juga memungkinkan anda untuk mengatur tombol Prt Scrn di pojok kanan pada kebanyakan papan ketik untuk mengambil cuplikan layar. Untuk melakukan hal ini, atur pengikat tombol lingkungan desktop anda supaya menjalankan xfce4-screenshooter dengan satu atau beberapa opsi berikut ketika tombol Prt Scrn ditekan.

Opsi baris perintah

Opsi -w

Opsi -w memungkinkan anda untuk mengambil cuplikan layar jendela yang aktif.

Opsi -f

Opsi -f memungkinkan anda untuk mengambil cuplikan layar seluruh layar.

Opsi -r

Opsi -r memungkinkan anda untuk memilih daerah yang akan ditangkap dengan cara klik pada titik di layar tanpa melepas tombol, lalu tarik tetikus anda ke sudut lain daerah, kemudian lepaskan tombol tetikus.

Opsi -d

Opsi -d yang diikuti integer positif memungkinkan anda untuk mengatur jeda sebelum mengambil cuplikan layar ketika opsi -f, -w atau -r diberikan.

Opsi -s

Opsi -s yang diikuti dengan alamat folder yang telah ada memungkinkan anda untuk mengatur dimana cuplikan layar akan disimpan. Opsi ini hanya berdampak jika opsi -f, -w atau -r diberikan. Jika dikombinasikan dengan opsi -h (lihat di bawah), opsi ini dapat digunakan untuk menyimpan berkas ke alamat tertentu, tanpa menampilkan dialog simpan apapun.

Opsi -h

Jika anda memberikan opsi -h, cuplikan layar akan disimpan tanpa menampilkan dialog simpan apapun. Opsi ini hanya berdampak jika opsi -f, -w atau -r diberikan. Opsi -s memungkinkan anda untuk mengatur lokasi penyimpanan jika diperlukan.

Opsi -o

Jika opsi -o diberikan, diikuti dengan nama aplikasi, cuplikan layar akan disimpan di direktori sementara sistem dan dibuka dengan aplikasi yang namanya disebutkan setelah -o. Opsi ini hanya berdampak jika opsi -f, -w, atau -r diberikan.

Menggunakan plugin untuk panel Xfce

Menambah plugin

Untuk menambah plugin ke panel, klik kanan pada panel dan pilih "Tambah Item Baru". Pilih plugin bernama "Cuplikan Layar", lalu tambahkan.

Mengatur plugin

Anda dapat mengatur bagaimana cuplikan layar diambil dengan klik kanan pada ekstensi lalu pilih "Konfigurasi". Dialog pengaturan yang serupa dengan dialog aplikasi utama akan muncul. Silakan lihat di atas untuk pengaturan yang berbeda.

Menggunakan plugin

Untuk mengambil cuplikan layar, cukup klik pada plugin.
